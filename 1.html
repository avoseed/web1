HTML

하이퍼텍스트 마크업 언어(Hypertext Markup Language, HTML)는 웹 페이지와 웹 애플리케이션을 만들기 위한 표준 마크업 언어입니다. 웹 브라우저는 웹 서버나 로컬 저장소로부터 HTML 문서를 받아 멀티미디어 웹 페이지로 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 기술하며, 원래는 문서의 겉모습에 대한 단서도 포함하고 있었습니다.

HTML 요소(element)는 HTML 페이지를 구성하는 기본 단위입니다. HTML 구성 요소를 통해 이미지나 대화형 폼 같은 다른 객체들을 렌더링되는 페이지 안에 삽입할 수 있습니다. HTML은 제목, 문단, 목록, 링크, 인용문 등 텍스트에 대한 구조적 의미를 부여함으로써 구조화된 문서를 만드는 수단을 제공합니다. HTML 요소는 꺾쇠 괄호(< >)를 사용해 작성하는 태그로 구분됩니다.
