CSS

종속형 시트(Cascading Style Sheets, CSS)는 HTML과 같은 마크업 언어로 작성된 문서의 표현 방식을 기술하는 데 사용되는 스타일 시트 언어입니다. CSS는 HTML, JavaScript와 더불어 월드 와이드 웹의 핵심 기술 중 하나입니다.

CSS는 표현(presentation)과 내용(content)의 분리를 가능하게 하도록 설계되었으며, 여기에는 레이아웃, 색상, 글꼴이 포함됩니다.[3] 이러한 분리는 콘텐츠 접근성을 향상시키고, 표현 특성을 지정하는 데 더 많은 유연성과 제어권을 제공하며, 관련 CSS를 별도의 .css 파일로 지정함으로써 여러 웹 페이지가 서식을 공유할 수 있게 하고, 구조적 콘텐츠의 복잡성과 반복을 줄여 줍니다.

서식과 내용의 분리는 동일한 마크업 페이지를 화면, 인쇄, 음성(음성 기반 브라우저나 스크린 리더를 통해), 점자 기반 촉각 장치 등 서로 다른 렌더링 방식에 맞춰 다른 스타일로 표현하는 것도 가능하게 합니다. 또한 CSS에는 모바일 기기에서 콘텐츠에 접근할 때를 위한 대체 서식 규칙도 있습니다.[4]

'종속형(cascading)'이라는 이름은, 특정 요소에 둘 이상의 규칙이 일치할 때 어떤 스타일 규칙을 적용할지 결정하는 우선순위 체계에서 유래했습니다. 이 종속 우선순위 체계는 예측 가능합니다.

CSS 명세는 월드 와이드 웹 컨소시엄(W3C)에서 관리합니다. 인터넷 미디어 타입(MIME 타입) text/css는 RFC 2318(1998년 3월)에 의해 CSS와 함께 사용하도록 등록되었습니다. W3C는 CSS 문서를 위한 무료 CSS 검증 서비스를 운영합니다.[5]

HTML 외에도 XHTML, 순수 XML, SVG, XUL 등 다른 마크업 언어들도 CSS 사용을 지원합니다.
